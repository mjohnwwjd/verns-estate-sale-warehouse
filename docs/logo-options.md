# Fresh Canva Logo Options

These five options were generated from scratch in Canva using the old Vern's signs only as style reference.

## Direction Given To Canva

- Brand name must be exactly `Vern's Estate Sale Warehouse`.
- Use singular `Estate Sale`, not `Estate Sales`.
- Keep the old spirit: yellow/gold letters, dark charcoal/black and gray sign feel, bold block type, decorative V energy.
- Do not copy the old logo or old decorative V exactly.
- Make it readable for the website header, storefront sign, shirts, and social profile.
- Avoid tiny text, thin script, trendy corporate gradients, generic thrift-store clip art, and shopping-cart icons.
